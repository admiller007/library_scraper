Key Features

  • Concurrent Scraping - Async operations reduce total time to ~2-3 minutes
  • Error Resilience - Individual failures don't crash entire pipeline
  • Retry Logic - Exponential backoff for network/rate limit errors
  • Data Quality - Text cleaning, deduplication, format validation
  • Multiple Outputs - CSV, PDF, ICS formats for different use cases
  • Web Interface - Real-time filtering and search capabilities
  • Automated Updates - Daily cron job on Render.com (6 AM)
